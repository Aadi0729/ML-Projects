How It Works

Introduction

The Student Performance Prediction project aims to predict students' academic performance based on various parameters. This machine learning project is designed to help educators, administrators, and policymakers make informed decisions to improve student outcomes. The application uses a machine learning model to analyze input data and provide predictions on student performance.

Features

- User-friendly interface for data input.
- Predicts student performance based on gender, race/ethnicity, parental education level, lunch type, test preparation course, and scores in writing and reading.
- Displays prediction results in a clear and concise manner.

Predict Data

The predict data page contains a form where users can input various details required for prediction. The form fields include:

  • Gender
  • Race or Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course
  • Writing Score
  • Reading Score

Upon form submission, the data is sent to the server for prediction.

Server-side Processing

The Flask application handles the form submission and processes the data. The data from the form is collected and transformed into a format suitable for the prediction pipeline. The CustomData class is used to structure the input data. The PredictPipeline class is used to load the trained machine learning model and make predictions. The prediction results are returned and rendered on the predict data page.

Prediction Pipeline

The prediction pipeline includes the following stages:

  • Data Ingestion: Collects and processes the input data from the form.
  • Data Transformation: Transforms the input data into a suitable format for the model.
  • Model Prediction: Uses the trained machine learning model to predict the student's performance based on the input data.

Displaying Results

The prediction results are displayed on the predict data page below the form after submission.
